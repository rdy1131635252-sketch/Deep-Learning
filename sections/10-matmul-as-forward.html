这是整个教程最关键的"顿悟时刻":你花时间学的矩阵乘法,就是神经网络的一切。 现在把线代和深度学习连起来。

核心等式

$$\Large \boxed{\,\mathbf{y} = f(W\mathbf{x} + \mathbf{b})\,}$$

一个神经网络层 = 三步:

  1. $W\mathbf{x}$ — 矩阵乘法(对输入做线性变换)
  2. $+\mathbf{b}$ — 向量加法(加偏置,整体平移)
  3. $f(\cdot)$ — 激活函数(逐元素作用的非线性,下一节详述)

一个具体的例子

设输入有 3 个特征(比如房子的[面积, 卧室数, 距市中心]),输出 2 个值(比如[预估价格, 热度]):

$$\mathbf{x} \in \mathbb{R}^3, \quad W \in \mathbb{R}^{2\times 3}, \quad \mathbf{b} \in \mathbb{R}^2, \quad \mathbf{y} \in \mathbb{R}^2$$
$$\begin{bmatrix}y_1\\y_2\end{bmatrix} = f\!\left( \begin{bmatrix}w_{11}&w_{12}&w_{13}\\w_{21}&w_{22}&w_{23}\end{bmatrix} \begin{bmatrix}x_1\\x_2\\x_3\end{bmatrix} + \begin{bmatrix}b_1\\b_2\end{bmatrix} \right)$$

展开第一个输出:$y_1 = f(w_{11}x_1 + w_{12}x_2 + w_{13}x_3 + b_1)$ —— 这就是一个神经元。矩阵乘法的威力在于一次性把所有神经元的计算都算了。

批量处理 — 真实训练的样子

实际训练时,我们不是一次喂一个样本,而是一次喂一个批次(batch),比如 64 个样本。 设 $X \in \mathbb{R}^{B \times d_{in}}$ 是一批 $B$ 个样本,每个 $d_{in}$ 维:

$$Y = f(X W^T + \mathbf{b})$$

形状推演:

$$\underbrace{X}_{B \times d_{in}} \cdot \underbrace{W^T}_{d_{in} \times d_{out}} + \underbrace{\mathbf{b}}_{d_{out}} = \underbrace{Y}_{B \times d_{out}}$$
关于 $W^T$ PyTorch 里 nn.Linear(in_features, out_features) 的权重是 $W \in \mathbb{R}^{d_{out}\times d_{in}}$。 调用时做 $X W^T$ 得到正确形状。记号上 $W$ 形状以"输出×输入"为主流,因为每一行对应一个神经元。

偏置是怎么加的?广播(Broadcasting)

$\mathbf{b}$ 是 $d_{out}$ 维向量,但 $XW^T$ 是 $B \times d_{out}$ 的矩阵。 它们相加时,$\mathbf{b}$ 会自动复制 $B$ 次 —— 每个样本用同一个偏置。这就是"广播"。

多层网络 = 矩阵乘法链

一个 3 层网络:

$$\mathbf{h}_1 = f_1(W_1\mathbf{x} + \mathbf{b}_1)$$ $$\mathbf{h}_2 = f_2(W_2\mathbf{h}_1 + \mathbf{b}_2)$$ $$\mathbf{y} = f_3(W_3\mathbf{h}_2 + \mathbf{b}_3)$$

一个 100 层的 Transformer = 100 次"矩阵乘法 + 非线性"的组合。训练就是调整所有 $W_i, \mathbf{b}_i$ 让 $\mathbf{y}$ 接近目标。

为什么必须有非线性 $f$?

假设我们去掉激活函数:

$$\mathbf{y} = W_3(W_2(W_1\mathbf{x} + \mathbf{b}_1) + \mathbf{b}_2) + \mathbf{b}_3$$

展开:

$$\mathbf{y} = \underbrace{(W_3 W_2 W_1)}_{W'}\mathbf{x} + \underbrace{(W_3 W_2 \mathbf{b}_1 + W_3 \mathbf{b}_2 + \mathbf{b}_3)}_{\mathbf{b}'}$$

!!! —— 它等于一个单层网络 $\mathbf{y} = W'\mathbf{x} + \mathbf{b}'$。 没有非线性,叠多少层都没用 —— 矩阵乘积还是矩阵。非线性 $f$ 才让多层真正"有用"。

通用近似定理 (Universal Approximation) 只要一个隐藏层、一个非线性激活、足够多的神经元,就能以任意精度逼近任意连续函数。

这个定理告诉我们"理论上能学任何东西",但没说"容易学"。实践中深(多层)比宽(单层很多神经元)更有效。

偏置 $\mathbf{b}$ 的几何意义

没有偏置时,$\mathbf{y} = f(W\mathbf{x})$ 的决策边界必过原点(因为 $\mathbf{x}=\mathbf{0}$ 时 $\mathbf{y}$ 固定)。 加上偏置就相当于允许决策边界平移 —— 这是线性变换(过原点)→ 仿射变换(任意位置)的区别。 没有这一点,网络就学不了大多数现实问题。

PyTorch 代码

import torch
import torch.nn as nn

# 定义一个"输入 3 维,输出 2 维"的全连接层
layer = nn.Linear(in_features=3, out_features=2)

# 查看参数形状
print("权重 W:", layer.weight.shape)   # (2, 3)  ← 输出×输入
print("偏置 b:", layer.bias.shape)     # (2,)

# 前向:单个样本
x = torch.tensor([1.0, 2.0, 3.0])
y = layer(x)                    # 内部做了 W @ x + b
print("输出:", y)

# 前向:批量(8 个样本)
X = torch.randn(8, 3)            # (batch=8, features=3)
Y = layer(X)                     # (8, 2)

# 加激活
import torch.nn.functional as F
Y_activated = F.relu(Y)
