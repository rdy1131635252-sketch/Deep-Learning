Transformer = 多个注意力层 + 前馈层堆起来。它取代 RNN 成为 NLP 默认骨架,后来扩散到 CV、语音、生物 —— 催生了 ChatGPT、Stable Diffusion、AlphaFold。这一节解剖它的每一块。

整体架构

原始 Transformer (Vaswani 2017) 是 Encoder-Decoder 结构,用于翻译。后来分化成三种范式:

类型组成代表模型典型任务
Encoder-only只用编码器BERT, RoBERTa, ViT分类、嵌入、理解
Decoder-only只用解码器(因果掩码)GPT, LLaMA, Claude生成、对话
Encoder-Decoder都有T5, BART, 原 Transformer翻译、摘要

Transformer Block(Pre-LN 现代版)

现代 Transformer(GPT-2 之后)采用 Pre-LN 设计 —— LayerNorm 在残差前(更稳定):

图 6.2.1 一个 Transformer Block · LN → Attn → 残差 · LN → FFN → 残差 · 一般堆 12-96 层。

位置编码(Positional Encoding)

注意力本身不知道顺序 —— 打乱输入顺序,输出也只是被打乱(置换不变)。 但语言显然有顺序!"狗咬人"≠"人咬狗"。所以要把位置信息加进 embedding:

正弦位置编码(原版)

$$PE_{(pos, 2i)} = \sin(pos / 10000^{2i/d}), \qquad PE_{(pos, 2i+1)} = \cos(pos / 10000^{2i/d})$$

不同维度对应不同频率的正余弦。优势:可以泛化到比训练长的序列。

RoPE(旋转位置编码)— 当今 LLM 主流

LLaMA、Qwen、Mistral 都用 RoPE:把位置作为旋转角度嵌入到 Q/K 里, 让点积自然包含相对位置信息。比加性 PE 在长上下文上效果更好。

FFN — 4× 扩展规则

$$\text{FFN}(x) = W_2 \cdot \text{GELU}(W_1 x + b_1) + b_2$$

中间层维度通常是输入的 4 倍(d_model = 512 → ffn_dim = 2048)。这是经验发现的甜蜜点。 现代 LLaMA 用 SwiGLU(三个矩阵 + Swish 门控),效果更好。

FFN 占多少参数? 整个 Transformer 的一大半参数在 FFN!不在注意力。这促成了 MoE(稀疏专家混合)优化: 让多组 FFN 共享存储,每次只激活其中一组。

完整数据流(Decoder-only,GPT 类)

  1. Tokenize:句子 → token IDs(BPE 分词)
  2. Embedding:每个 ID 查一个向量,加上位置编码
  3. N 层 Transformer Block:每层做注意力 + FFN
  4. 最后 LN + 线性投影到 vocab_size
  5. Softmax → 下一个 token 的概率分布
  6. 采样(贪心 / top-k / top-p)→ 输出 token,作为下一步输入(自回归)

KV Cache — 推理加速的关键

生成第 100 个 token 时,前 99 个的 K, V 已经算过了。缓存它们,新一步只算第 100 个的 Q, 和已有的 K, V 算注意力即可。无 KV cache:每生成 1 token = 重算整个序列;有 KV cache:每步是常数操作。

Transformer 家族树

类型代表特点
NLP-EncoderBERT, RoBERTa双向理解,适合分类
NLP-DecoderGPT 系列, Claude, LLaMA生成,因果掩码
NLP-BothT5, BART序列到序列任务
视觉ViT, Swin, DINO图像切 patch 当 token
多模态CLIP, BLIP, GPT-4V图文对齐
生物AlphaFold2, ESM-2蛋白质结构、功能
扩散DiT, Stable Diffusion条件生成主干

PyTorch 代码

import torch.nn as nn

class TransformerBlock(nn.Module):
    def __init__(self, d_model=512, n_heads=8, ffn_dim=2048):
        super().__init__()
        self.ln1 = nn.LayerNorm(d_model)
        self.attn = nn.MultiheadAttention(d_model, n_heads, batch_first=True)
        self.ln2 = nn.LayerNorm(d_model)
        self.ffn = nn.Sequential(
            nn.Linear(d_model, ffn_dim),
            nn.GELU(),
            nn.Linear(ffn_dim, d_model),
        )

    def forward(self, x, mask=None):
        # Pre-LN: norm 在残差之前
        a, _ = self.attn(self.ln1(x), self.ln1(x), self.ln1(x), attn_mask=mask)
        x = x + a
        x = x + self.ffn(self.ln2(x))
        return x

# PyTorch 内置
encoder = nn.TransformerEncoder(
    nn.TransformerEncoderLayer(d_model=512, nhead=8, batch_first=True),
    num_layers=6
)
