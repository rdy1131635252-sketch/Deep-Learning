梯度下降需要梯度。怎么算?答案是链式法则,从后往前传播 —— 反向传播。 它让训练百层神经网络变成可能。

核心:链式法则的三段色

对一个权重 $w$,损失 $L$ 的梯度分成三段:

$$\frac{\partial L}{\partial w} = \underbrace{\frac{\partial L}{\partial \hat{y}}}_{\color{red}{损失}} \cdot \underbrace{\frac{\partial \hat{y}}{\partial z}}_{\color{purple}{激活}} \cdot \underbrace{\frac{\partial z}{\partial w}}_{\color{blue}{线性}}$$
  • 损失函数 的导数(最后一步)
  • 激活函数 的导数
  • $z = wx+b$ 对 $w$ 的偏导 = $x$

计算图(Computation Graph)

神经网络的任何运算都可以画成有向无环图:节点 = 运算,边 = 数据流动。 PyTorch / TensorFlow 的 Autograd 就是基于这种图:

  1. 前向:每个节点算一遍,存下输入和输出
  2. 反向:从终点损失往前,每个节点按它的"局部导数规则"把上游梯度乘一下往下游传

三种基本节点的梯度规则

节点类型前向反向
加法 $y = a + b$ 就是相加 梯度原样分发:$\frac{\partial L}{\partial a} = \frac{\partial L}{\partial y}$
乘法 $y = a \cdot b$ 就是相乘 梯度互换:$\frac{\partial L}{\partial a} = \frac{\partial L}{\partial y} \cdot b$
max $y = \max(a, b)$ 较大者 梯度路由到 argmax(另一个为 0)

ReLU 就是 max(0, x):正区梯度 = 1 路由到下游,负区梯度 = 0 被截断。

雅可比矩阵视角

对向量输入/输出的层 $\mathbf{h} = f(\mathbf{z})$,"局部导数"是雅可比矩阵 $J$:

$$J_{ij} = \frac{\partial h_i}{\partial z_j}$$

反向传播的本质:

$$\underbrace{\frac{\partial L}{\partial \mathbf{z}}}_{\text{传给下游}} = J^T \underbrace{\frac{\partial L}{\partial \mathbf{h}}}_{\text{上游来的梯度}}$$

每一层都是"上游梯度 × 局部雅可比"。 工程实现里几乎不显式算完整 $J$,而是用 vector-Jacobian product (VJP),省大量内存。

PyTorch 自动求导

import torch

# 前向
x = torch.tensor([1.0, 2.0], requires_grad=True)
w = torch.tensor([0.5, -0.3], requires_grad=True)
b = torch.tensor(0.1, requires_grad=True)

z = (w * x).sum() + b       # 线性组合
y = torch.sigmoid(z)         # 激活
L = (y - 1.0) ** 2           # MSE 损失

# 一行反向传播 — 自动算完所有梯度
L.backward()

print("∂L/∂w =", w.grad)
print("∂L/∂b =", b.grad)

# 注意:下一轮训练前要 opt.zero_grad()
# 不然梯度会累加!
Autograd 的魔法 你只写前向代码,框架通过计算图自动推导反向传播,得到所有参数的梯度。 这是现代深度学习框架的核心能力,也是研究者能快速实验各种新架构的基础。
