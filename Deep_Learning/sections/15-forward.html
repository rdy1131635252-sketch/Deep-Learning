前向传播 = 矩阵乘 + 加偏置 + 激活 的层层堆叠。 这是神经网络最核心的计算过程。

步步演示

固定 $\mathbf{x}=[0.5, 0.8]$ 和一组权重,点"下一步"观察数据如何流过网络:

互动 3.5 数据从左到右,依次穿过权重矩阵。

代码实现

import torch

# 3层网络: 2 → 2 → 1
W1 = torch.randn(2, 2)   # 输入→隐藏
b1 = torch.zeros(2)
W2 = torch.randn(1, 2)   # 隐藏→输出
b2 = torch.zeros(1)

x = torch.tensor([0.5, 0.8])

# ── 前向 ──
z1 = x @ W1 + b1          # 线性变换
a1 = torch.sigmoid(z1)    # 激活
z2 = a1 @ W2.T + b2       # 再线性变换
y  = torch.sigmoid(z2)    # 再激活
print(y)                  # tensor([0.5249...])

矩阵维度对应

运算形状含义
$\mathbf{x}$$(1, 2)$一个样本,两个特征
$\mathbf{W}_1$$(2, 2)$输入2 → 隐藏2
$\mathbf{z}_1$$(1, 2)$隐藏层线性输出
$\mathbf{a}_1$$(1, 2)$隐藏层激活
$\mathbf{W}_2$$(1, 2)$隐藏2 → 输出1
$\mathbf{\hat{y}}$$(1, 1)$最终预测
批量前向 把 $\mathbf{x}$ 从 $(1,2)$ 换成 $(64,2)$ —— 一行代码,64个样本同时算完。底层就是矩阵并行的魔力!
