SVD 是线性代数里最万用的工具:对任何矩阵(方的、长的、胖的、奇异的)都能做。 推荐系统、图像压缩、降噪、伪逆、PCA —— 它们背后都是 SVD。

核心公式

$$A_{m\times n} = U_{m\times m}\,\Sigma_{m\times n}\,V^T_{n\times n}$$
几何直觉 任何线性变换 = 旋转 → 缩放 → 再旋转。SVD 正好把这三步分离:
  • $V^T$:在输入空间里旋转(正交矩阵,只转不缩)
  • $\Sigma$:沿各个正交主轴缩放(对角矩阵,对角线上的 $\sigma_i \geq 0$ 叫"奇异值")
  • $U$:在输出空间里再旋转(正交矩阵)
图 1.7.1 SVD 的三步几何分解:任何矩阵都是"转 → 缩 → 再转"。

与特征分解的对比

特征分解 $A = PDP^{-1}$SVD $A = U\Sigma V^T$
适用只对可对角化的方阵任何矩阵
分解唯一?相似变换不唯一基本唯一(奇异值唯一)
正交?一般 $P$ 不正交$U, V$ 都是正交的
数值稳定不稳定(病态矩阵难做)非常稳定

特殊情形关系:若 $A$ 是对称正定方阵,则特征分解 = SVD(取 $U = V$)。

计算流程(2×2)

  1. 计算 $A^T A$(对称,一定可以特征分解)
  2. $A^T A$ 的特征值 $\lambda_i$ ≥ 0,奇异值 $\sigma_i = \sqrt{\lambda_i}$
  3. $A^T A$ 的特征向量(正交、归一化)→ 组成 $V$
  4. $\Sigma$ 是 $\sigma_i$ 的对角阵
  5. $U = AV\Sigma^{-1}$(对非零奇异值)

低秩近似 — SVD 的杀手锏

SVD 按奇异值大小"分层":$A = \sigma_1 \mathbf{u}_1\mathbf{v}_1^T + \sigma_2\mathbf{u}_2\mathbf{v}_2^T + \dots$ 每一项是"秩-1"的矩阵。只保留最大的 $k$ 项,得到最好的秩-$k$ 近似:

$$A_k = \sum_{i=1}^{k}\sigma_i\mathbf{u}_i\mathbf{v}_i^T$$

这叫 Eckart-Young 定理:$A_k$ 是所有秩-$k$ 矩阵中最接近 $A$ 的(Frobenius 范数意义下)。

真实数据:奇异值衰减很快

一张 1000×1000 的自然图像,奇异值从大到小排列通常前 50 个占总能量的 95%+。 于是:

  • 原始数据:$10^6$ 个像素值
  • 秩-50 近似:$2 \times 1000 \times 50 + 50 = 10^5$ 个数 → 压缩 10 倍,视觉几乎无损

应用

  • 图像压缩:JPEG 之外的另一条路线
  • 推荐系统:Netflix 奖比赛 — "用户×电影" 评分矩阵做 SVD,用低秩近似补全缺失评分
  • 去噪:小奇异值往往对应噪声,扔掉重构
  • PCA:对中心化数据做 SVD ⇔ PCA(第 1.8 节)
  • 伪逆:$A^+ = V\Sigma^+ U^T$
  • LoRA:大模型微调用低秩更新 $\Delta W = BA$,是 SVD 思想在深度学习的直接体现

NumPy 代码

import numpy as np

A = np.array([[3, 2], [2, 1]])
U, sigma, Vt = np.linalg.svd(A)

print("U =\n", U)
print("奇异值 =", sigma)     # 向量,不是对角阵
print("V^T =\n", Vt)

# 重建 A
A_back = U @ np.diag(sigma) @ Vt
print("重建:\n", A_back)     # 约等于 A

# 秩-1 近似(只保留最大的奇异值)
A_rank1 = sigma[0] * np.outer(U[:, 0], Vt[0])
print("秩-1 近似:\n", A_rank1)

# 伪逆
A_pinv = np.linalg.pinv(A)
