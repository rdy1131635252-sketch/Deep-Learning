PCA 是机器学习最经典的降维算法。本质是对数据协方差矩阵做特征分解 —— 它把"抽象的特征向量"变成了直接可视化的"数据的主方向"。

核心思想

直觉 看一个斜着拉长的数据云:数据主要沿长轴方向变化,短轴方向变化很小。 PCA 就是要自动找出方差最大的几个正交方向,把数据投影到这些方向上,从而保留主要信息降低维度。

流程

  1. 中心化:减去均值,$\tilde{X} = X - \bar{X}$
  2. 协方差矩阵:$C = \frac{1}{n-1}\tilde{X}^T\tilde{X}$(对称!)
  3. 特征分解:$C = V\Lambda V^T$
  4. 排序取前 k:按特征值从大到小,取前 $k$ 个特征向量作为主成分方向 $V_k$
  5. 投影:$Y = \tilde{X} V_k$,$Y$ 是 $k$ 维的降维结果

为什么"方差最大 = 主特征向量"

这是 PCA 的核心洞察,用拉格朗日乘子法推导。我们要找 $\mathbf{w}$ 使投影方差 $\mathbf{w}^T C \mathbf{w}$ 最大,约束 $\|\mathbf{w}\|=1$:

$$\mathcal{L} = \mathbf{w}^T C \mathbf{w} - \lambda(\mathbf{w}^T\mathbf{w} - 1)$$

对 $\mathbf{w}$ 求导并置零 $\Rightarrow 2C\mathbf{w} - 2\lambda\mathbf{w} = 0$,即:

$$C\mathbf{w} = \lambda\mathbf{w}$$

完美! 方差最大的方向,就是协方差矩阵 $C$ 最大特征值对应的特征向量。 方差值 = 特征值 $\lambda$。次大方差方向 = 第二大特征值对应的向量,依此类推。

动手试试

互动 1.8 红色长线 = PC1(方差最大方向),橙色短线 = PC2。拉高相关性,看 PC1 变得多主导。

解释方差比 · 怎么选 k

第 $i$ 个主成分解释了多少方差:

$$\text{explained variance ratio}_i = \frac{\lambda_i}{\sum_j \lambda_j}$$

实际中常用手肘法:画出累积解释方差 vs k 的曲线,找"拐点"。或者直接取累积 90% 或 95% 时的 k。

与 SVD 的关系

PCA = SVD on 中心化数据 $\tilde{X} = U\Sigma V^T$,那么协方差 $\frac{1}{n-1}\tilde{X}^T\tilde{X} = \frac{1}{n-1}V\Sigma^2 V^T$。 所以 $V$ 的列向量就是 PCA 的主成分,$\sigma_i^2 / (n-1)$ 就是对应特征值。

实务中常直接对 $\tilde{X}$ 做 SVD,比先算协方差再分解更稳定。

应用

  • 数据可视化:1000 维嵌入 → 2/3 维画出来看
  • 去噪:次要主成分往往是噪声,只保留主成分 = 去噪
  • 加速训练:特征降维后模型更小
  • Eigenfaces(人脸识别的经典方法):每张脸 = 平均脸 + 一组"特征脸"的加权组合
局限 PCA 只能发现线性结构。如果数据分布是非线性流形(如螺旋、瑞士卷),PCA 会失败。 此时应该考虑 t-SNE、UMAP(降维可视化)、Kernel PCA(非线性版 PCA)、 或神经网络的自编码器。

sklearn 代码

from sklearn.decomposition import PCA
import numpy as np

# X: (n_samples, n_features)
X = np.random.randn(100, 50)

pca = PCA(n_components=2)       # 降到 2 维
Y = pca.fit_transform(X)         # (100, 2)

# 每个主成分解释了多少方差
print(pca.explained_variance_ratio_)
# 比如 [0.31, 0.18] → 前两个主成分一起解释了 49% 的方差

# 累积解释
import matplotlib.pyplot as plt
pca_full = PCA().fit(X)
plt.plot(np.cumsum(pca_full.explained_variance_ratio_))
# 看拐点选 k
