处理图像时,全连接网络参数爆炸、丢失空间信息。CNN 优雅地解决了这些 —— 它引入参数共享和局部连接,让网络能在大图上高效学习。

什么是"卷积"?

卷积 = 用一个小矩阵(卷积核/滤波器)在大图上滑动,每个位置做逐元素相乘再求和。这就是在提取特征。

$$\text{Output}_{ij} = \sum_m \sum_n \text{Input}_{i+m,\,j+n} \cdot \text{Kernel}_{mn}$$

为什么 CNN 完爆全连接(处理图像时)

全连接CNN
参数量(224×224 图)每层数百万参数每层几千参数
空间结构拉成长向量,丢失保留 2D 结构
参数共享无同一核扫全图
平移不变性无有(猫在图左右都能识别)

关键超参数

  • 核大小(kernel_size):3×3、5×5 最常见。多个 3×3 堆叠可以替代一个 5×5 / 7×7,参数更少、非线性更多
  • 步幅(stride):每次滑动几格。stride=2 → 输出尺寸减半
  • 填充(padding):边缘补 0,防止输出缩小。same padding 让输出尺寸 = 输入
  • 通道数(channels):彩色图输入 3 通道,中间层可有几十~几百通道
$$\text{输出尺寸} = \left\lfloor\frac{\text{输入} + 2p - k}{s}\right\rfloor + 1$$

多通道卷积

彩色图输入 (3, H, W),经过一个卷积层,输出 $(C_{out}, H', W')$。核的形状是 $(C_{out}, C_{in}, k, k)$:

一句话理解多通道 每个输出通道 = $C_{in}$ 个单通道卷积的结果相加。
所以 32 个 3×3 卷积核对 3 通道输入 → 参数量 = $32 \times 3 \times 3 \times 3 = 864$(加偏置 32)。

感受野(Receptive Field)

感受野 = 当前层的一个神经元"看到"原始输入的区域大小。每多一层卷积,感受野变大 —— 深层神经元能看到更大范围的原图。

$$RF_{l} = RF_{l-1} + (k_l - 1) \cdot \prod_{i=1}^{l-1} s_i$$

例子:三个 3×3 卷积(stride=1)堆叠,感受野 = 3 → 5 → 7。两个 3×3 卷积的感受野 = 一个 5×5 的感受野,但参数更少且非线性更多 —— 这就是 VGG 设计的灵魂。

1×1 卷积 — 近乎免费的通道混合器

1×1 卷积核在空间上不做任何聚合(只盖一个像素),但在通道维度做全连接:

  • 把 $C_{in}$ 通道降维或升维到 $C_{out}$
  • 计算量 = 普通 $k\times k$ 卷积的 $1/k^2$
  • 是 Inception 模块、ResNet 瓶颈、MobileNet 的关键组件
