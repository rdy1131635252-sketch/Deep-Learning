CNN 在过去 25 年走了一条"更深、更巧妙、更高效"的路。下面每个经典架构都用 SVG 画出来,看完你对深度学习视觉的演化就心中有数了。

池化层(Pooling)

池化 = 把一个小区域压缩成一个值。最大池化取最大,平均池化取平均。 作用:降低分辨率、减少参数、扩大感受野、提供平移不变性。

$$\text{MaxPool}_{2\times2}\begin{bmatrix}1&3\\5&2\end{bmatrix} = 5, \quad \text{AvgPool}_{2\times2} = \frac{1+3+5+2}{4} = 2.75$$

ResNet 之后,池化逐渐被 stride=2 的卷积 替代 —— 卷积是可学习的。 最后一层常用全局平均池化(GAP)代替全连接层,大幅减少参数。

1. LeNet-5 (1998) — CNN 鼻祖

Yann LeCun 发明,用于手写数字识别。结构:2 卷积 + 2 池化 + 3 全连接,证明卷积是处理图像的正确工具。

图 4.2.1 LeNet-5 · 约 6 万参数,已能在 MNIST 上达 99%。

2. AlexNet (2012) — 引爆深度学习

Krizhevsky, Sutskever, Hinton 的 AlexNet 在 ImageNet 上把 top-5 错误率从 26% 打到 16%,震动学界,开启深度学习时代。创新:更深(8 层)+ ReLU + Dropout + GPU + 数据增强。

图 4.2.2 AlexNet · 8 层,~6000 万参数,首个真正"深度"的 CNN。

3. VGG (2014) — 简洁极致

Oxford VGG 组:只用 3×3 小卷积堆叠 + 2×2 池化,越深越好。VGG-16 = 13 卷积 + 3 全连接。结构极简,参数巨大(约 1.38 亿),但泛化特征好,至今是迁移学习常用 backbone 之一。

图 4.2.3 VGG-16 · 5 个"conv-pool"块 + 3 全连接。宽度随深度翻倍。

4. GoogLeNet / Inception (2014)

Google 团队作品,2014 ImageNet 冠军。核心是 Inception 模块:同层并行多尺度卷积(1×1, 3×3, 5×5)再拼接。1×1 降维是关键,极省算力。

图 4.2.4 Inception 模块:同层多条路并行,最后拼接。

5. ResNet (2015) — 深度学习的奇点

Kaiming He 作品,2015 ImageNet 冠军。核心创新 —— 残差连接:让某些层直接把输入加到输出,$y = F(x) + x$。梯度能"抄近道"回传,彻底解决了超深网络难训的问题。ResNet-152(152 层!)成功训出,准确率刷新纪录。

图 4.2.5 ResNet 基本单元。近十年最重要的结构创新之一,Transformer 也继承了这个设计。

6. MobileNet (2017) — 移动端利器

Google 为手机设计的 CNN。核心创新:深度可分离卷积 —— 把标准卷积拆成两步:深度卷积(每通道独立 3×3,不混通道)+ 逐点卷积(1×1 混通道)。算力从 $K^2 C_{in} C_{out}$ 降到 $K^2 C_{in} + C_{in} C_{out}$,通常省 $8\sim 9$ 倍。

图 4.2.6 MobileNet 核心:把"混空间"和"混通道"解耦。

7. EfficientNet (2019)

Google 用神经架构搜索(NAS)找到基础架构 EfficientNet-B0,然后按复合缩放规则同比放大深度、宽度、分辨率,得到 B1-B7 系列。每一个都在相同算力下准确率最高。

图 4.2.7 EfficientNet · 平衡缩放 > 单维缩放。

8. Vision Transformer / ViT (2020)

Google 把 Transformer 从 NLP 搬到视觉:把图像切成 patches,每个 patch 当作一个"词",喂进标准 Transformer 编码器。大数据预训练后准确率超越 CNN —— 新一代视觉基座。

局部性不是必要的,数据够大时 Transformer 全局注意力比 CNN 表达力更强
图 4.2.8 ViT · 把 CV 彻底 Transformer 化。

PyTorch 迷你 CNN

import torch.nn as nn

class MiniCNN(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()
        self.features = nn.Sequential(
            nn.Conv2d(1, 32, kernel_size=3, padding=1),
            nn.BatchNorm2d(32),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(2),                         # 28→14
            nn.Conv2d(32, 64, kernel_size=3, padding=1),
            nn.BatchNorm2d(64),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(2),                         # 14→7
        )
        self.classifier = nn.Sequential(
            nn.Flatten(),
            nn.Linear(64*7*7, 128),
            nn.ReLU(inplace=True),
            nn.Dropout(0.5),
            nn.Linear(128, num_classes),
        )

    def forward(self, x):
        return self.classifier(self.features(x))

model = MiniCNN()
x = torch.randn(8, 1, 28, 28)   # (batch, channel, H, W)
out = model(x)                    # (8, 10)
